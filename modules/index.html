Modules in Es6

  • Video Source.
  • Notes: Module is a file that export some code. Basically it acts as your custom lib file.
  • Must add attribute type="module" on the script tag to define it as a module
  • Keyword import is to import the external file(with its component stuff)
  • Keyword export is to allow the function/variable/classes to able to be imported in the other js file
